### For use this project...

##### install dependence
```
Composer install
```
##### Create database
```
php bin/console doctrine:database:create
```
##### database migration
```
php bin/console make:migration
```
##### create shemas databse
```
php bin/console doctrine:schema:create
```
##### load fixtures
```
php bin/console doctrine:fixture:load
```
